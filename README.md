# Cadastro de Clientes

Projeto prático focado em treinar a lógica de programação e entender o consumo de APIs linha por linha. Desenvolvido como exercício do curso de Desenvolvedor Full Stack Python (EBAC).

# Tecnologias Utilizadas
* HTML & CSS
* JavaScript 
* API CrudCrud (Backend mockado)

## Como rodar o projeto
1. Gere uma nova URL no site [CrudCrud](https://crudcrud.com/).
2. Substitua o valor da variável `API_URL` no arquivo `scripts.js` com o seu link gerado.
3. Abra o arquivo `index.html` no seu navegador usando o Live Server.
